
## 🚀 使用方式

### 开发模式
```bash
# 安装依赖
npm install

# 启动Electron应用
npm start
# 或
start.bat

# 使用Node.js命令行版本
node AutoGAME.js [选项]
# 或
run-nodejs.bat [选项]
```

### 生产使用
```bash
# 构建便携版
npm run dist
# 或
build.bat

# 自动运行模式
auto-run.bat
```
